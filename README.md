This repository reflects my ongoing journey in mastering PowerShell for enterprise environments, including Active Directory, Windows Server, cloud integrations, and general automation tasks.

## Purpose

This repo serves as:

- A reference for common admin tasks
- A place to document my learning and progression in PowerShell
- A showcase of practical automation techniques used in real IT environments
