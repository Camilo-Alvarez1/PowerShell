## 🧰 Tech Stack

A collection of the tools, shells, and environments used throughout this repository:

- **Windows PowerShell 5.1**  
  The classic Windows‑integrated engine used for legacy modules, Hyper‑V management, and AD administration.

- **PowerShell 7 (pwsh)**  
  Cross‑platform, modern PowerShell used for improved performance, updated modules, and cleaner scripting patterns.

- **PowerShell ISE**  
  Used for quick testing, debugging, and legacy script editing in Windows environments.

- **Visual Studio Code**  
  Primary development environment with extensions for PowerShell, Git, and automation workflows.

- **Windows Server (2019–2025)**  
  Lab environment for Active Directory, DNS/DHCP, Group Policy, and Hyper‑V automation.

- **Hyper‑V**  
  Virtualization platform used for VM provisioning, auditing, and lab automation.

- **Git & GitHub**  
  Version control, documentation, and script publishing.
